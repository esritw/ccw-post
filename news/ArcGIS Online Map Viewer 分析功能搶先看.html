ArcGIS Online Map Viewer 分析功能搶先看

14 四月, 2023阿牛哥

ArcGIS Online Map Viewer是一個快速、直觀且靈活的線上地圖編輯器。它使用最新的Web GIS技術 — ArcGIS API for JavaScript 4.x，因此擁有先進且高效的地理空間資料視覺化功能。隨著尖端功能不斷地被發佈出來，我們幾乎每天都會看到一個問題： Map Viewer何時提供分析工具？對此，我們有公告一個令人興奮的消息。隨著本次ArcGIS Online的更新，在Map Viewer中已經可以使用分析工具了！

新的分析體驗

在Map Viewer中，有一種新的方式可以使用分析。首先，從右側工具欄中選擇「分析」，即可打開分析窗格。分析窗格有兩個選項：「工具」和「歷程記錄」。

分析工具

Map Viewer中的工具包括圖徵分析工具和網格分析工具(網格分析功能需要有ArcGIS Image for ArcGIS Online的User Type才能使用)。您可以根據分析類型來篩選分析工具，或透過關鍵字搜尋找到相關的工具。與Map Viewer Classic一樣，您仍然可以瀏覽並從同一類別中選擇分析工具。請注意，您可以根據工具圖示，區分工具是用於「圖徵分析」或是「網格分析」。

我們在Map Viewer中增加了更多關於工具的詳細說明，讓它們更容易使用。例如，必要的參數會標有紅點。圖層增加後，您可以看到有多少筆圖徵紀錄會被分析。透過新的環境設定，您可以為分析結果的輸出圖層指定投影，而不必與輸入圖層相同。

我們也讓您更容易在運行工具之前估算所需的點數。Map Viewer中的分析工具消耗的點數與Map Viewer Classic中的相同。

隨著2023年2月的首次發佈， Map Viewer已經涵蓋了使用者在Map Viewer Classic中，經常使用的圖徵分析工具。下面的影片展示了如何在 Map Viewer中使用追蹤下游(Trace Downstream)和豐富圖層(Enrich Layer)工具，預測熔岩流動並估計潛在影響。

請注意，Map Viewer Classic中的一些工具已在Map Viewer中更名，以方便新用戶找到。例如，在Map Viewer Classic中，「Create Drive-Time Areas」工具現在在Map Viewer中稱為「Generate Travel Areas」。請參閱此說明以獲取最新的工具更名清單和Map Viewer支援的工具列表。

對於在Map Viewer中使用ArcGIS Image擴充套件的用戶，您現在可以使用此版本中提供的24個網格分析工具；不過，網格函數和網格函數鏈編輯器目前在Map Viewer中尚未得到支援。未來版本中，卓越的分析功能將遷移到Map Viewer中。請閱讀「What’s New in ArcGIS Image for ArcGIS Online 」以獲取更多資訊。

分析歷史記錄

在 Map Viewer 中，分析歷史記錄是一個令人興奮的新功能。對於那些熟悉資料分析和資料科學的人來說，他們知道分析是一個反覆運行的過程。分析人員經常需要多次重新運行相同的分析程序，以更改參數、查看警告和錯誤等。在 Map Viewer 中，每次您在 Web 地圖中運行工具時，關於該工具及其參數的訊息都會保存在歷史記錄中。所有的工具運行都會被增加到分析歷史記錄中，包括成功運行、失敗或取消的作業。您可以查看有關每個運行作業的詳細訊息。例如，工具生成的結果、工具何時完成、運行工具所需的時間、任何警告或錯誤消息以及工具的輸入參數，包括環境設定。您可以使用相同或修改過的參數重新打開和運行歷史中的任何工具。以下影片展示了如何使用分析歷史記錄簡化工作流程，以研究在不同交通狀況下消防局的響應時間。

儲存 Web 地圖後，分析歷史記錄也會與 Web 地圖一起存儲，並可與組織中的任何人共享。分析歷史記錄可幫助您更好地了解用於建立 Web 地圖項目中的資料和圖層的工作流程。此外，它是將您的線上分析過程與他人共享的一種絕佳方式。

現在就可以開始使用 Map Viewer 中的分析工具了！請記住，Map Viewer 中還提供了圖表，因此您可以使用資料探勘分析和視覺化分析來探索資料，以及用於解釋和傳達分析結果的圖表。在未來一年中，我們還將繼續增加更多的工具和功能。

像往常一樣，我們非常樂意聽取您的經驗和意見！

祝您分析愉快！
